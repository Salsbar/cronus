A set of scripts to fully automate Dungeons & Bombs. 
Sequences taken from: https://psnprofiles.com/guide/11809-dungeons-bombs-trophy-guide.

AutoPlay1-50.axb: 
There's a chance that the first script Autoplay1-50 will complete the entire game. 
However, there appears to be some RNG with level 45 so it may fail.

AutoPlay45.axb
If level 45 fails during Autoplay1-50, use this script. 

AutoPlay46-50.axb
Continue on using this script to finish the game.
