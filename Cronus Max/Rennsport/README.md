# Game
Rennsport - Maxxxxed out! (Reach Account Level 80)

# General Information
As of March 2026, Rennsport single player championships award 100 xp for immediate disqualifications. 
This script automates the loop of:
1. Restarting the Rookie BMW M2 championship.
2. Skipping through practice and qualification rounds.
3. Getting a quick DSQ by driving backwards.

Note that it may be necessary to have an existing save in the Rookie BMW M2 championship so the script sequence doesn't get messed up. However, once you do this step once, it won't be necessary again.
