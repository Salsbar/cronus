Game: Heads Up! Phones Down Edition

Achievement(s):
Heads Up! Novice     (10 Different Decks)
Heads Up! Apprentice (25 Different Decks)
Heads Up! Genius     (50 Different Decks)
Heads Up! Master     (90 Different Decks)

Setup:
Set the game time to 30s in the settings. 
Be sure that 'All Decks' is selected on the left side menu. 
Manually start up the first deck. 
After the 30s deck has ended, it will be on the post-match score screen. 
Start the script here.
